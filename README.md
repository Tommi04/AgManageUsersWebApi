Progetto dimostrativo full-stack per la gestione utenti con operazioni CRUD.

## Panoramica
Questo repository raccoglie un piccolo esercizio realizzato con:
- **Angular** per il front-end
- **ag-Grid** per la visualizzazione e modifica tabellare
- **ASP.NET Web API** per il back-end REST
- **SQL Server** con **Entity Framework Migrations** per persistenza e versionamento schema

L'obiettivo è mostrare una pipeline completa di gestione utenti: creazione, lettura, aggiornamento e cancellazione.

## Contenuto del repository
Al momento il codice sorgente applicativo è incluso negli archivi:
- `AgUsers.7z`
- `WebApiUsers.7z`

## Funzionalità principali
- Elenco utenti in tabella con ag-Grid
- Inserimento nuovi utenti
- Modifica dati utente
- Eliminazione utenti
- Backend API per operazioni CRUD

## Note
Si tratta di un progetto didattico/prototipale sviluppato in poche ore per esercizio.
